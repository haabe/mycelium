# Skills by category

**Audience**: practitioners scanning skills by type of work.
**Time to read**: 3 min.
**Last updated**: 2026-05-08.

Alternate index to the phase-first [skills/README.md](README.md). Same 49 skills, different ordering.

## Research & discovery

User research, interviews, evidence-gathering, classification. Use when you do not yet know what you are deciding.

- `/user-interview` тАФ story-based interviewing (Torres)
- `/mocked-persona-interview` тАФ disciplined mocked personas when real users unavailable
- `/user-needs-map` тАФ needs independent of solutions (Allen)
- `/jtbd-map` тАФ Jobs to be Done (Christensen)
- `/cynefin-classify` тАФ domain classification (Snowden)
- `/wardley-map` тАФ strategic landscape (Wardley)
- `/handoff` тАФ structured offline conversation prep
- `/log-evidence` тАФ record findings back into canvas

## Synthesis & planning

Turn research into decisions. Use when discovery has produced enough signal to converge.

- `/ost-builder` тАФ Opportunity Solution Tree (Torres)
- `/ice-score` тАФ Impact ├Ч Confidence ├Ч Ease (Ellis)
- `/gist-plan` тАФ GIST planning (Gilad)
- `/assumption-test` тАФ smallest viable test (Torres + Gilad AFTER)
- `/diamond-progress` тАФ move a diamond through gate checks
- `/diamond-assess` тАФ current state + recommended next action

## Build & delivery

Implementation, validation, completion. Use when scope is committed and you are building.

- `/delivery-bootstrap` тАФ auto-detect tech stack + generate validation
- `/preflight` тАФ pre-implementation checklist
- `/reflexion` тАФ implement тЖТ validate тЖТ self-critique тЖТ retry
- `/definition-of-done` тАФ executable completion gates
- `/dora-check` тАФ DORA metrics (Forsgren)

- `/service-check` тАФ 15 service principles (Downe)
- `/threat-model` тАФ STRIDE

- `/security-review` тАФ OWASP Top 10:2025

## Evidence & metrics

- `/metrics-detect` тАФ find applicable metric sources
- `/metrics-pull` тАФ snapshot + compute deltas + draft evidence

## Orchestration

- `/fan-out` тАФ parallel agent orchestration
- `/canvas-update` тАФ write evidence to canvas
- `/canvas-health` тАФ lint canvas

- `/launch-tier` тАФ go-to-market classification (Lauchengco)
